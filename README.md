# IMDB Clone

Full-stack movie database web app with a React frontend and Node.js/MongoDB backend. Browse movies, view details, and manage a watchlist.

**Stack**

React · MUI · Node.js · MongoDB · Axios · Formik · React Router

**Setup**

```bash
git clone https://github.com/chiragbagde/imdb-clone.git
cd imdb-clone

# Backend
cd backend
npm install
npm start

# Frontend (new terminal)
cd imdb-clone
npm install
npm start
```

Requires a local MongoDB instance running on the default port.
